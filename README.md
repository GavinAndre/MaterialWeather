# materialweather

基于Material Design设计规范的一个天气应用

功能如下：

(1)多城市管理
(2)卡片式界面（今日天气详情，生活指数建议，昨日今日后四天天气）
(3)可点击切换简介与详情卡片页面

使用ApiStore的天气查询接口。
使用数据库来存储天气城市等信息。
界面使用RecycleView,CardView,TabLayout,CollapsingToolbarLayout等基于MaterialDesign的新控件和布局
